## [CHECK IT OUT](https://fav-lyric.netlify.app/)

### Fav-lyric! This app is a place to store and log your favorite lyrics! You can also check out other users favorite lyrics!

### Technologies Used
  - HTML
  - CSS
  - JavaScript
  - TypeScript
  - Sequalize
  - Bit.io
  - Fly.io
  - PostgreSQL
